Tribonacci Numbers

The Tribonacci Sequence is similar to the well-known Fibonacci Sequence, but instead of the first two terms being 0 and 1, the first three terms are 0, 0, and 1. After this, each successive term is defined as the sum of the three terms preceding it. The first few terms in the sequence are 0, 0, 1, 1, 2, 4, 7, 13, 24, 44, 81, 149, 274, 504, 927, 1705, 3136, 5768, 10609, 19513, 35890... which can be seen at this link (oeis.org).
